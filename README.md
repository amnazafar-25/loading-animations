# ⏳ CSS Loading Animations

A collection of clean and creative loading animations built using pure HTML and CSS. This project showcases various animated loaders like spinners, dots, ripples, and wave bars — perfect for improving UI feedback and enhancing UX on modern websites.

> 💼 **Created as part of my front-end development portfolio projects.**

## 🚀 Live Demo

🔗 [View Project Live](https://amnazafar-25.github.io/loading-animations/)

## 🧰 Technologies Used

- HTML5  
- CSS3 (Animations, keyframes, transforms)

## 🎯 Animations Included

- 📶 **Bar Wave Loader** — vertical wave bars bouncing in sequence  
- 💧 **Dots Loader** — three bouncing dots with smooth timing  
- 🌊 **Ripple Loader** — circular ripple effect with expanding rings  
- 🔄 **Spinner Loader** — rotating ring spinner (classic style)

## 📁 Project Structure

loading-animations/
├── index.html
├── barwave.css
├── dots.css
├── ripple.css
└── spinner.css


Each loader has its own CSS file and can be reused independently in any project.

## 📌 How to Use

1. Open the [live demo](https://amnazafar-25.github.io/loading-animations/)  
2. Browse through different CSS-only loading animations  
3. Copy the relevant HTML + CSS code for your own use
